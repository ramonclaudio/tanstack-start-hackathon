# tanstack-start-hackathon

My submission for the TanStack Start hackathon. SaaS starter combining TanStack Start, Convex, Better Auth, Autumn billing, and Sentry. I kept iterating on it after the hackathon, so the active version lives at [`tanvex`](https://github.com/ramonclaudio/tanvex). This repo is the original snapshot.

SaaS starter: TanStack Start + Convex + Better Auth + Autumn billing + Sentry.

## If you want to use this

Clone [`tanvex`](https://github.com/ramonclaudio/tanvex) instead. It has the latest fixes, cleaner structure, and updated deps.

## Stack

- TanStack Start (SSR, Router, Query)
- React 19
- Tailwind CSS 4 + shadcn/ui + Radix
- Convex (real-time DB, auth sessions)
- Better Auth (email + GitHub OAuth)
- Autumn (Stripe billing, Free/Starter/Pro tiers)
- Sentry (error tracking)
- Netlify (edge hosting)
- Bun, Vite, TypeScript, Vitest
